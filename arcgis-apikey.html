ALICE Data Explorer

Enter your ArcGIS API key to access the mapping features. This bypasses the need for OAuth login and supports automatic authentication.

How to get your API key:

  1. Go to developers.arcgis.com
  2. Sign in with your ArcGIS account (supports 2FA)
  3. Navigate to your dashboard
  4. Click on "API Keys" or create a new key
  5. Copy the key and paste it above

Your API key is stored locally and never sent to any server.
